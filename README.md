# Age Certification App

A small JavaScript-based web project that asks the user for their date of birth and checks if they meet the age requirement of 15 years or older.

## Purpose

This project was built to:

- Practice **JavaScript date and age calculations**
- Use **DOM input handling** and **form validation**
- Learn **conditional logic** and **user redirection**

---

## Features

- Takes three user inputs: Day, Month, Year
- Calculates the user's current age
- Shows an alert if the age is under 15: `"Your age is restricted"`
- Redirects to Google if age is **15 or above**

---

## Technologies Used

- HTML5 for structure
- CSS3 for minimal styling
- JavaScript for logic and interaction
